# Boilerplate
__Build the project according to the [responsive design](https://www.figma.com/file/k6TUmzltDOVhfZR3D9XUeI/Dev_project).__

## Getting Started

#### Install dependencies:

```yarn
yarn
```
#### Run on dev mode
```yarn
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `pages/index.js`. The page auto-updates as you edit the file.

### Run on production mode
```yarn
yarn build
yarn start
```

## Combined Linter

#### Check

```yarn
yarn lint
```

#### Fix

```yarn
yarn lint:fix
```


## Separated Linters

#### Eslint check

```yarn
yarn eslint:check
```

#### Eslint fix

```yarn
yarn eslint:fix
```

#### Stylelint check

```yarn
yarn stylelint:check
```

#### Stylelint fix

```yarn
yarn stylelint:fix
```

#### Prettier check

```yarn
yarn prettier:check
```

#### Prettier fix

```yarn
yarn prettier:fix
```

## Unit Tests

```yarn
yarn test
```

#### Run with coverages

```yarn
yarn test:coverage
```

[API routes](https://nextjs.org/docs/api-routes/introduction) can be accessed on [http://localhost:3000/api](http://localhost:3000/api). This endpoint can be edited in `pages/api/index.js`.

The `pages/api` directory is mapped to `/api/*`. Files in this directory are treated as [API routes](https://nextjs.org/docs/api-routes/introduction) instead of React pages.
